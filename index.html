Quem Somos

O projeto Speaking Engish Kids foi elaborado por alunos da disciplina Projeto Integrador do período letivo 2024.2 do curso de Análise e Desenvolvimento de Sistemas da faculdade Unifacisa, com identidade visual própria para o projeto. O principal objetivo foi a criação de um site para o ensino de inglês para crianças.

Programa

O Speaking English Kids é um programa que concentra outras plataformas que podem auxiliar no aprendizado de inglês projetado especialmente para crianças em idade escolar. Com uma metodologia lúdica, interativa e adaptada às necessidades de cada etapa de aprendizado, nosso objetivo é facilitar aos usuários a terem uma experiência divertida e envolvente.O spinking inglish Kiss é separado
